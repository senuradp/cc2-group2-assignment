## How to setup

Run the following commands to setup the project:

- git clone "name"
- composer install
- create .env file from .env.example
- php artisan key:generate
- npm install
- git checkout "your branch name"
- php artisan migrate
- php artisan db:seed
- php artisan serve

Your good to go now!
